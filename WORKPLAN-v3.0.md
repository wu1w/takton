# Takton v3.0 — 工作计划

> 日期：2026-07-13
> 周期：4 周（20 个工作日）
> 范围：MCP 客户端接入、桌面原生体验增强、Skill/Tool 体系整合
> 排除：多 Agent 编排与本地 SLM 推理（v4.0）

---

## 总策略

1. **先统一内部工具抽象（D）**：不统一，MCP 和 Skill 都接不进来。
2. **再接入 MCP（A）**：这是 v3.0 最大价值点，也是工具生态入口。
3. **最后做桌面原生体验（C）**：依赖 Electron 主进程，和前两步耦合较少，可并行。
