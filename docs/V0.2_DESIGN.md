## 1. Desktop Agent 层（最高优先级）

### 1.1 目标
让 Takton 能直接操作用户桌面（点击、输入、打开应用、文件管理），实现真正的"Computer Use"。

### 1.3 平台适配

#### Windows（主平台）
- **方案**: [Windows-MCP](https://github.com/CursorTouch/Windows-MCP)（CursorTouch 开源）
- **原理**: Windows UI Automation (UIA) API，非坐标级，精准识别控件
- **能力**:
  - `open_app(name)` - 打开应用
  - `click(element_id)` - 点击按钮/链接
  - `type(element_id, text)` - 输入文本
  - `screenshot()` - 获取屏幕 + UIA 元素树
  - `scroll(direction, amount)` - 滚动
  - `drag(from, to)` - 拖拽

#### Mac
- **方案**: [MacOS-MCP](https://github.com/CursorTouch/MacOS-MCP)（同团队）
- **原理**: macOS Accessibility API
- **能力**: 与 Windows 对齐

#### Linux
- **方案**: Terminal + Screenshot + xdotool
- **降级策略**: 
  - 优先尝试 `xdotool`（X11）
  - 无 GUI 环境时仅支持 terminal 命令执行
  - 截图用 `import` (ImageMagick) 或 `scrot`

### 1.5 权限系统

#### 三级权限模式

| 级别 | 行为 | 存储 |
|------|------|------|
| **Allow Once** | 仅本次操作，下次再问 | 不存储 |
| **Allow Session** | 本次会话内不再询问 | 内存，重启清空 |
| **Always Allow** | 永久不再询问 | 数据库 `desktop_permissions` 表 |

### 1.6 安全策略

```python
# backend/services/desktop/security.py

SAFE_APPS = {
    "notepad.exe", "code.exe", "chrome.exe", 
    "excel.exe", "word.exe", "explorer.exe"
}

SENSITIVE_OPERATIONS = {
    "delete_file", "format_disk", "modify_registry",
    "install_software", "uninstall_software"
}

def require_confirmation(app_name: str, operation: str) -> bool:
    """判断是否需要用户确认"""
    if operation in SENSITIVE_OPERATIONS:
        return True
    if app_name not in SAFE_APPS:
        return True
    return False
```

```python
async def _run_agent_loop(self, ...):
    # 初次 RAG 检索
    rag_results = await self.rag_service.search(query, top_k=3)
    
    # 判断检索质量
```

### 4.2 并行执行器

```python
# backend/agent/cluster_executor.py (新增)

import asyncio
from typing import List, Dict

class ClusterExecutor:
    """集群并行执行器"""
    
    async def execute(
        self, 
        task: str, 
        sub_agent_ids: List[str]
    ) -> ClusterResult:
        # 1. 协调者分析任务
        coordinator = CoordinatorAgent()
        plan = await coordinator.plan(task, sub_agent_ids)
        
        # 2. 并行执行子任务
        tasks = []
        for sub_task in plan.sub_tasks:
            agent = self._get_agent(sub_task.agent_id)
            tasks.append(
                self._run_sub_agent(agent, sub_task)
            )
        
        results = await asyncio.gather(*tasks, return_exceptions=True)
        
        # 3. 聚合结果
        aggregator = AggregatorAgent()
        final_result = await aggregator.aggregate(results, plan)
        
        return final_result
    
    async def _run_sub_agent(self, agent, sub_task):
        """运行单个子代理"""
        # 独立的 messages 上下文
        messages = [
            {"role": "system", "content": agent.system_prompt},
            {"role": "user", "content": sub_task.description}
        ]
        
        # 调用 LLM
        result = await self.llm_service.chat(messages)
        
        return SubAgentResult(
            agent_id=agent.id,
            task=sub_task,
            result=result,
            status="completed"
        )
```

```python
            "detail": str(exc),
            "traceback": traceback.format_exc()
        }
    )
```
